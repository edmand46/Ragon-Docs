### Ragon Unity SDK

### Installing

After project creation add Ragon Unity SDK package

Open Unity Package Manager, press "+", press on "Add package from git URL"

![img.png](img/upm_git_url.png)

and past next:

```
https://github.com/edmand46/Ragon-Unity-SDK.git
```
